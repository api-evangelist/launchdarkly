# LaunchDarklyThis is a repo for managing machine readable artifacts for the [LaunchDarkly APIs](http://launchdarkly.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [LaunchDarkly APIs](http://launchdarkly.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://launchdarkly.com)- [Developer](http://launchdarkly.com)- [Twitter](https://twitter.com/LaunchDarkly)- [GitHub](https://github.com/launchdarkly)The [APIs.json](https://github.com/api-evangelist/launchdarkly/blob/master/apis.json) index provides links to all the relevant links for the LaunchDarkly APIs, including the OpenAPI and Postman if they are present.
